Exportar Datos

Para exportar los datos al portapapeles por favor permita el acceso de esta página al portapapeles.

Una vez copiados los datos, envialos al dispositivo donde realizarás la importación.


Otra opción es descargar los datos como archivo para realizar la importación.

Una vez descargado el archivo ábrelo en el dispositivo donde realizarás la importación y copias los datos.
